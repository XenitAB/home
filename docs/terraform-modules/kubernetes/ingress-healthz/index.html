Ingress Healthz (ingress-healthz)

This module is used to deploy a very simple NGINX server meant to check the health of cluster ingress. It is meant to simulate an application that expects traffic through the ingress controller with automatic DNS creation and certificate creation, without depending on the stability of a dynamic application.
