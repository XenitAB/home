GitHub Actions Self-hosted Runner

This is the setup for GitHub Actions Self-hosted Runner Virtual Machine Scale Set (VMSS).

GitHub Runner Configuration#

Setup a GitHub App according to the documentation for XenitAB/github-runner.

Setup an image using Packer according github-runner in XenitAB/packer-templates.
